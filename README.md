# 🤖 Mochi AI Chatbot  
### CODSOFT Artificial Intelligence Internship – Task 1

Mochi AI is a **voice-based AI assistant** built using Python.  
It can listen to your voice commands, process them, and respond using speech while displaying an animated robot interface.

---

## 🚀 Features

🎤 Voice Input (Speech Recognition)  
🔊 Voice Response (Text to Speech)  
🤖 Animated Robot Interface  
📚 Knowledge from Wikipedia  
🌐 Google Search Integration  
⚡ Fast Response System  

---

## 🧠 Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- CustomTkinter
- Pillow
- Wikipedia API

## ⚙️ Installation

### 1️⃣ Install Python
Download Python 3.10 from:

https://www.python.org/downloads/

---

### 2️⃣ Install Required Libraries

Run this command in terminal:


pip install customtkinter pillow SpeechRecognition pyttsx3 pyaudio wikipedia


---

## ▶️ How to Run

Navigate to project folder and run:


python main.py


The **Mochi AI interface will open** and start listening for commands.
