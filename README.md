# CODSOFT AI Internship

## Task 1 - Mochi AI Chatbot 🤖

Mochi AI is a voice-based AI assistant built using Python.

### Features
- Voice input using microphone
- Voice response using text-to-speech
- Animated robot interface
- Wikipedia knowledge integration
- Google search support

### Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- CustomTkinter
- Wikipedia API

### How to Run

1. Install Python 3.10

2. Install required libraries

pip install customtkinter pillow SpeechRecognition pyttsx3 pyaudio wikipedia

3. Run the project

python main.py
